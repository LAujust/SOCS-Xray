Quick Start
=======================

This is a quick guidance for using the SOCS_Xray to systematically search Optical Counterparts. 


Installation
-----------------------

- From Source

.. code-block:: bash

    git clone https://github.com/LAujust/SOCS-Xray.git

    cd EPTools/

    pip install .

- From pip (under assessment)

.. code-block:: bash

    pip install mypackage
